¿Que es HTML?

Es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos.

¿Que es CSS?

Las hojas de estilo en cascada ( CSS , cascading style sheets) permiten crear páginas web atractivas.

¿Que es JavaScript?

Es un lenguaje de programación diseñado en un principio para añadir interactividad a las páginas webs y crear aplicaciones web. A pesar de la similitud en el nombre, no está relacionado con Java.
